## Say How Are You

| title             | description                         | source                                         | keywords             |
| ----------------- | ----------------------------------- | ---------------------------------------------- | -------------------- |
| say "how are you" | Say How Are You for the given names | [https://localhost/how](https://localhost/how) | how, say-how-are-you |

```js
var names = ["John", "Bob"];
for (var i in names) {
    console.log(`Hey ${names[i]}! How are you?`);
}
```
